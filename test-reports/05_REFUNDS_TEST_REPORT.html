Feature Overview: The Refund Requests page displays all customer refund requests sorted by date. Each request shows: Order ID, Customer email, Reason (Changed Mind, Item Damaged, Wrong Item, Not As Described, Other), truncated detail message, Order total, current Status badge, and action buttons. Admin can add optional notes and click Approve (green) or Deny (red). After action, buttons are replaced with "—" and the status badge updates.

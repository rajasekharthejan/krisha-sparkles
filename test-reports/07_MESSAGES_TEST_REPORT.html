Feature Overview: The Messages page displays all customer contact form submissions. Each message shows: Sender name, email, subject category badge (Order Issue, Return Request, Product Question, General, Other), date/time, truncated message, and three action buttons: Expand/Collapse (eye icon), Mark Read / Mark Unread (toggles), and Reply (opens mailto: link in email client). Unread messages have a gold dot indicator and slightly highlighted background.

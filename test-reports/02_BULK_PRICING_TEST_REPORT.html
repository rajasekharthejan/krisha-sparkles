Feature Overview: Bulk Pricing allows admins to update prices for multiple products simultaneously. Admin selects products by category or search, chooses an operation (Increase / Decrease) with a modifier (Percent / Fixed Amount), enters a value, previews the calculated new prices in a table, then confirms to apply changes to the database.
